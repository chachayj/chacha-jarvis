# chacha-jarvis-be-flask

frontend/web 의 html들을 호스팅하기 위한 flask 프레임워크를 사용한 backend

# 언어
Python 3.12.4

# venv (for MacOs)
```
python3 -m venv venv

-- 활성화

source venv/bin/activate

-- 비활성화 

deactivate

```

# 1. install requirments
```
pip3 install -r requirements.txt
```

# 2. run
```
python3 ./src/app.py
```

 * Running on http://127.0.0.1:5000
